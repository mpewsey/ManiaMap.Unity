LayoutState  layoutState,

Sets the current layout and layout state to the manager and initializes it.

Parameters
layoutThe layout.
layoutStateThe layout state.
settingsThe manager settings. If null, the settings will fallback to the first valid of the following:
  1. The existing manager settings, if they are not null.
  2. The settings loaded from the resources path, if it exists.
  3. A newly initialized settings object.
Exceptions
LayoutIsNullExceptionRaised if the layout is null.

System.ArgumentExceptionRaised if the layout and layout state's ID's do not match.
